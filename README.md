# LUMIÈRE KYIV

![LUMIÈRE KYIV Preview](./public/favicon.png)

## 🍽️ About The Project

**LUMIÈRE KYIV** is a luxury fine dining restaurant website created as a modern frontend portfolio project.

The goal of the project was to build an elegant and immersive digital experience for a premium restaurant brand, focusing on sophisticated UI design, smooth animations, responsive layouts, and clean frontend architecture.

The website presents the restaurant concept, signature dishes, menu, gallery, and reservation experience through a refined luxury interface.

---

## ✨ Features

- 🌙 Premium dark luxury design
- 📱 Fully responsive layout for all devices
- 🍷 Restaurant presentation sections
- 📖 Interactive menu showcase
- 🖼️ Elegant image gallery
- 📅 Table reservation form
- ✅ Form validation
- ✨ Smooth scrolling animations
- 🔝 Scroll to top functionality
- 🌍 SEO optimized metadata
- ⚡ Fast loading with Vite

## 🚀 Getting Started

### Clone repository

```bash

git clone https://github.com/BohdanKrav4enko/LUMIERE.git
```

### Install dependencies

```bash

pnpm install
```

### Run development server

```bash

pnpm dev
```

### Build project

```bash

pnpm build
```

### Preview production build

```bash

pnpm preview
```
